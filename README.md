## PyTorch Scholarship Challenge

After being awarded one of the 10,000 seats in the **Facebook AI & PyTorch Scholarship Challenge**, I was fortunate to be selected, together with approximately 300 fellow students worldwide, to continue into the **Udacity Deep Learning Nanodegree**.

The scholarship program was created through a partnership between **Facebook AI** and **Udacity** with the goal of helping learners acquire advanced deep learning skills using **PyTorch**, one of the leading open-source deep learning frameworks.

The program combined theoretical foundations with hands-on implementation across several major areas of artificial intelligence, including:

* Neural Networks
* Computer Vision
* Natural Language Processing
* Generative Adversarial Networks (GANs)
* Model Deployment
* Production-Oriented AI Workflows

Students learned from leading researchers and practitioners including:

* Sebastian Thrun
* Ian Goodfellow
* Jun-Yan Zhu
* Andrew Trask

## Learning Progression

This repository documents a progression through several major deep learning paradigms.

```text
Feedforward Neural Networks
            ↓
Computer Vision (CNNs)
            ↓
Sequence Models (RNNs / LSTMs)
            ↓
Generative Adversarial Networks
            ↓
Model Deployment
```

Together, these projects provide practical experience across a broad spectrum of modern deep learning techniques.

## Projects Included

| Project                                 | Focus Area                    | Technologies             |
| --------------------------------------- | ----------------------------- | ------------------------ |
| 🚲 Predicting Bike Sharing Patterns     | Neural Networks & Forecasting | Feedforward Networks     |
| 🐶 Dog Breed Classifier                 | Computer Vision               | CNNs & Transfer Learning |

---

### 🚲 Predicting Bike Sharing Patterns

Build and train a neural network capable of forecasting bicycle rental demand from historical data.

**Concepts**

* Backpropagation
* Gradient Descent
* Regression
* Forecasting
* Neural Network Fundamentals

---

### 🐶 Dog Breed Classifier

Develop a computer vision pipeline capable of detecting dogs and classifying dog breeds using transfer learning.

**Concepts**

* CNNs
* Transfer Learning
* Image Classification
* Feature Extraction
* Deep Learning for Computer Vision

---

### 📺 Generate TV Scripts

Train a recurrent neural network to generate synthetic television dialogue.

**Concepts**

* RNNs
* LSTMs
* Sequence Modelling
* Language Generation
* Neural Language Models

---

### 🎨 Generate Faces

Train a Generative Adversarial Network capable of synthesizing realistic human faces.

**Concepts**

* GANs
* Generative Modelling
* Adversarial Learning
* Image Synthesis
* Latent Representations

---

### 😊 Deploying a Sentiment Analysis Model

Build, train, and deploy an NLP model capable of classifying movie reviews as positive or negative.

**Concepts**

* Sentiment Analysis
* PyTorch
* Amazon SageMaker
* Model Deployment
* Inference Pipelines
