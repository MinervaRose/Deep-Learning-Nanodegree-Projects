## Overview

This project explores one of the foundational problems in machine learning:

**predicting future behaviour from historical data.**

Using a dataset of bicycle-sharing usage records, a neural network is trained to predict future rental demand based on historical patterns and environmental variables.

The project focuses on implementing and training a feedforward neural network while developing an understanding of the core mechanisms underlying modern deep learning systems.

Although simple compared to later architectures such as CNNs, GANs, or large language models, projects like this provide the foundations upon which more advanced AI systems are built.
